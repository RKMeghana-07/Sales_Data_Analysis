Sales Data Analyzer using Pandas

Internship: Syntecxhub - Data Science
Week 1 Project: Pandas CSV reader and basic analysis

This project reads supermarket sales data using Pandas, performs basic data analysis and exports filtered results.

Things learnt and did:
- Loaded sales CSV into a DataFrame using 'read_csv'  
- Displayed dataset insights using 'head()', 'tail()', 'shape', 'columns', 'info()'  
- Summary statistics using 'describe()'  
- Calculated business metrics: Total Revenue, Average Bill, Highest Bill,and Lowest Bill  
- Filters performed: High-value orders and Orders from a specific city
- Groupby analysis performed: Total sales by city and Total sales by product line
- Exported results into 3 CSV files(attached above).
